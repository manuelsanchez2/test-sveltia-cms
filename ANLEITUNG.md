# Anleitung: Inhalte der Website pflegen

Diese kurze Anleitung zeigt, wie Sie die Texte, Bilder und Kontaktdaten Ihrer Website
selbst ändern können — ganz ohne technische Kenntnisse.

## 1. Anmelden

1. Öffnen Sie **ihre-domain.de/admin** im Browser.
2. Klicken Sie auf **„Login with GitHub“** und melden Sie sich mit Ihrem GitHub-Konto an.
3. Sie sehen nun links die Bereiche **Einstellungen**, **Seiten** und **Rechtliches**.

## 2. Etwas ändern

1. Wählen Sie links den passenden Bereich, z. B. **Einstellungen → Allgemein** für
   Telefon, Adresse und Sprechzeiten.
2. Ändern Sie die gewünschten Felder.
3. Klicken Sie oben rechts auf **„Save“** (Speichern).

> **Gut zu wissen:** Ihre Änderungen werden zunächst auf einer **Vorschau-Seite**
> sichtbar, nicht sofort öffentlich. So können Sie in Ruhe prüfen, bevor etwas live geht.

## 3. Wo finde ich was?

| Ich möchte ändern … | Bereich im CMS |
|---|---|
| Telefon, Adresse, Sprechzeiten, Notdienst | Einstellungen → Allgemein |
| Instagram / Doctolib / jameda-Links | Einstellungen → Social Media |
| Text auf der Startseite | Seiten → Startseite |
| Liste der Leistungen | Seiten → Leistungen |
| Team-Mitglieder und Fotos | Seiten → Team |
| Texte & Fotos „Die Praxis“ | Seiten → Die Praxis |
| Anfahrt / Kontakt | Seiten → Kontakt & Anfahrt |
| Impressum / Datenschutz | Rechtliches |

## 4. Ein Foto hochladen

1. Klicken Sie im jeweiligen Bild-Feld auf **„Choose an image“**.
2. Ziehen Sie Ihr Foto in das Fenster oder wählen Sie es vom Computer.
3. Bild auswählen → **Save**.

_Tipp:_ Verwenden Sie möglichst Querformat-Fotos in guter Auflösung (mind. 1200 px breit).

## 5. Änderungen veröffentlichen (live schalten)

Nach dem Speichern sind die Änderungen auf der **Vorschau-URL** sichtbar.
Wenn alles passt, wird die Freigabe von uns (oder per Klick in GitHub) übernommen —
dann erscheinen die Änderungen auf der öffentlichen Website.

> Möchten Sie diesen letzten Schritt selbst übernehmen können? Sagen Sie uns Bescheid,
> wir richten dafür gerne einen einfachen „Veröffentlichen“-Knopf ein.

## Fragen?

Bei Unsicherheiten einfach nichts überschreiben und uns kurz kontaktieren —
es kann nichts kaputtgehen, jede Änderung lässt sich rückgängig machen.
